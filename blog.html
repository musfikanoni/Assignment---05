Important FAQ

what is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

how do you select an element from the DOM?

getElementById: For single elements by ID. getElementsByClassName & getElementsByTagName: For collections of elements. querySelector & querySelectorAll: For more flexible selections using CSS selectors.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated.
